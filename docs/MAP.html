Carte Interactive de la Pollution en Occitanie

À propos de cette carte

Cette carte interactive présente les données de pollution atmosphérique à travers la région Occitanie. Elle permet de visualiser les concentrations de différents polluants mesurés par les stations de surveillance de la qualité de l'air.

Comment utiliser cette carte :

  • Points colorés : Chaque marqueur représente une station de mesure. La couleur indique le niveau de pollution.
  • Zones rouges : Indiquent des concentrations élevées de polluants.
  • Cliquez sur un marqueur pour voir les détails des mesures.
  • Utilisez les contrôles pour filtrer par type de polluant ou période.

Légende :

Faible pollution (bonne qualité de l'air)
Pollution modérée
Pollution élevée
Très forte pollution (qualité de l'air dangereuse)

Polluants mesurés :

Les principaux polluants suivis incluent les particules fines (PM2.5, PM10), le dioxyde d'azote (NO₂), l'ozone (O₃) et le dioxyde de soufre (SO₂). Ces données sont collectées mensuellement par les stations de surveillance régionales.

Note : Les données présentées sont collectées auprès des stations de surveillance officielles. Les valeurs élevées peuvent indiquer des dépassements des seuils réglementaires de qualité de l'air.
